How it works

1 · Drop a document

PDF, scan, image, or a whole folder. Decant renders, detects layout, recovers tables, OCRs scans — all locally.

2 · Your Claude extracts

Define a schema (or let AI propose one). Your own Claude returns strict JSON — every value cites the block it came from.

3 · Verify & export

Click a field → its box lights up on the page. Fix low-confidence values. Export JSON/CSV/Excel, or write back into the form.

Not a wrapper

Real local pipeline

Render → layout → table structure → OCR → block model → validate → cite → export. The model does one step; your machine does nine.

Verifiable citations

Values reference block IDs that resolve to exact bounding boxes. Hallucinated citations are flagged, not trusted.

Works offline

Parse, OCR and export need no model at all. Connect Claude only for the smart extraction step.
